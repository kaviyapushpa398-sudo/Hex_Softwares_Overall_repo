# Hex_Softwares_Overall_repo

☕ Java Programming Internship Projects – Hex Softwares

📌 Overview

This repository contains all the projects I completed during my 1-month Java Programming Internship at Hex Softwares. The internship was designed to strengthen core Java skills through real-world applications. Although only 3 projects were mandatory, I challenged myself and successfully completed all 9 projects, gaining deeper practical experience.

---

🚀 Technologies Used

- Java (Core Java)
- OOP (Object-Oriented Programming)
- Collections Framework
- Exception Handling
- JDBC (for database integration in selected projects)

---

📂 Projects & Detailed Explanation

🔹 1. Travel Booking System

Developed a system to manage booking of flights, hotels, and transportation. Implemented structured classes to handle user input, availability, and booking confirmation. This project helped me understand how multi-service platforms work and improved my system design skills.

---

🔹 2. Inventory Management System

Built an application to manage products and inventory transactions. Included features like adding, updating, and deleting products along with stock tracking. Learned how to maintain real-time data consistency and handle records efficiently.

---

🔹 3. Employee Leave Management System

Created a workflow-based system where employees can request leave and admins can approve or reject requests. Implemented logic for leave balance tracking, improving my understanding of business process automation.

---

🔹 4. Hospital Management System (Java + JDBC)

Designed a database-connected system to manage patient records and hospital data. Used JDBC for database connectivity and performed CRUD operations. This project gave me hands-on experience with persistent data storage.

---

🔹 5. URL Shortener

Developed a tool to convert long URLs into shorter ones and redirect users. Implemented logic for unique code generation and mapping, which enhanced my understanding of efficient data handling.

---

🔹 6. Movie Database System

Built a system to manage movie records including adding, searching, and displaying data. Focused on organizing data and implementing search functionality.

---

🔹 7. Simple Banking System

Implemented core banking operations like account creation, deposit, withdrawal, and balance checking. Added error handling to prevent invalid transactions, improving my understanding of secure logic implementation.

---

🔹 8. Library Management System

Developed a system to manage books and members, including issuing and returning books. Implemented tracking features to monitor issued books and user records.

---

🔹 9. Student Grade Calculator

Created a system to calculate student grades, averages, and pass/fail status. Included functionality to handle multiple students and compute class performance.

---

💡 Key Learnings

- Strong foundation in Java and OOP concepts
- Practical experience in building real-world applications
- Hands-on knowledge of JDBC and database operations
- Improved problem-solving and debugging skills
- Understanding of system design and workflow implementation
